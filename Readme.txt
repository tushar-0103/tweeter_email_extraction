Welcome to Tweeter- Email Search 

Objective:-
	The primary objective of this project is to find emails in the tweets which are mentioned with the hashtags on tweeter.

Library used:
 Pandas 
 snscrape  (pip install snscrape)

How to use:

	simply run the program in python to get the output :

	you have to give two inputs:
		
			1) hashtags that you want to search
			2) number of tweets that are related to that hashtag 

Output:
  You will get emails which are mentioned in tweets related to that hashtag.
